### 📚 Project Settings
#### 📢 Install npm dependencies

```
$ npm install
```
#### 📢 Start dev-server

```
$ npm start
```

#### 📢 Run tests

```
// watch test
$ npm test

// unit test
$ npm run test:unit
```

#### 📢 Run build project

```
$ npm build
```

#### 📢 Run json-server

```
$ npm run server:start
```

#### 📢 Run Lint

```
$ npm run lint
```

#### 📢 Run Coverage

```
$ npm run coverage
```
